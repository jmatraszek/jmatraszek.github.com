## Install prerequisities

```
cargo install cobalt-bin
cargo install just
cargo install ghp
sudo aura -A nodejs-hackmyresume
git clone https://github.com/jmatraszek/jsonresume-theme-slick
cd jsonresume-theme-slick
sudo npm update
sudo npm install -g
```

## Build

```
just build
```

## Deploy

```
just deploy
```

## Clean

```
just clean
```
